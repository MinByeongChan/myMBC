# 자료형

String: UTF-8 문자열
ID: String, 식별자를 나타냄
Int 정수
Float 부호가있는 부동소수점 값
Boolean 참 또는 거짓 ( true, false)

`!` : 반드시 포함되어야하는 값을 의미. 없으면 에러가 발생하게 됨
특정 타입에 대해 null을 감지하고 싶을때 추가하면 된다.
